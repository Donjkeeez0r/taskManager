____
# Простой менеджер задач в командной строке с поддержкой событий и уведомлений.
____
##  :clipboard: Функционал:
- Добавление задач с заголовком и описанием
- Просмотр списка всех задач
- Отметка задач как выполненных
- Удаление задач

## :paperclip: Особенности:
- Реализована система уведомлений через EventEmitter
- Автоматическое сохранение в tasks.json

## :computer: Использование
```
# Добавить задачу
node index.js add "Купить молоко" "Не забыть взять сдачу"

# Показать все задачи
node index.js list

# Отметить задачу выполненной
node index.js done 1234567890

# Удалить задачу
node index.js delete 1234567890

```
