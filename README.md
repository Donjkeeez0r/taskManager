____
# Небольшой REST API сервер на чистом Node.js для управления задачами (To-Do менеджер).
____
## :pencil2: Возможности:
- Получение списка всех задач
- Создание новой задачи
- Обновление текста и статуса задачи
- Удаление задачи
- Валидация входных данных
- Хранение данных в JSON-файле
## :file_folder: Структура проекта
```
├── public/
│   └── index.html     # Клиентская часть
├── tasks.json         # Файл хранения задач
└── server.js          # HTTP сервер
```
